This repo contains a single-page React + Three.js test page for an interactive 3D text / dial visualization.

## Run locally

1. Install dependencies

```bash
npm install
```

2. Start dev server

```bash
npm run dev
```

Then open the URL printed in the terminal (usually `http://localhost:5173`).

## Build

```bash
npm run build
npm run preview
```

## Sharing with a client

- For a quick live link, this repo is set up to deploy to GitHub Pages via GitHub Actions.

### GitHub Pages (Actions) live link

1. Push this project to a GitHub repo (default branch: `main`).
2. In GitHub: **Settings** -> **Pages** -> **Build and deployment** -> select **GitHub Actions**.
3. Push (or re-push) to `main`. The workflow will build and deploy.

Your URL will be:

`https://<your-username-or-org>.github.io/<repo-name>/`
